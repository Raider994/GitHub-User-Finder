## Описание
GUI-приложение для конвертации валют с использованием актуальных курсов из внешнего API. Приложение позволяет сохранять историю конвертаций, импортировать/экспортировать данные в формате JSON.

## Требования
- Python 3.7+
- PyQt5
- requests
