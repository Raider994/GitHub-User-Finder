# GitHub User Finder

Это простой сайт для поиска пользователей GitHub по имени и добавления их в избранное.  
Реализован с помощью HTML, CSS и JavaScript. Всё хранится в локальном хранилище браузера.

---

## Описание

Этот проект позволяет:
- Вводить имя пользователя GitHub
- Получать из API GitHub информацию о пользователе
- Смотреть краткую информацию и добавлять пользователя в список избранных
- Сохранять избранных пользователей между сессиями в localStorage
